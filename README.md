# KOSS Hackathon — 맞춤형 메뉴 추천 서비스

> Flutter 기반 모바일 앱과 Node.js 백엔드로 구성된  
> **질문 기반 추천 시스템 + LLM 연계 재추천 구조**를 구현한 프로젝트입니다.  

---

## 프로젝트 개요
본 프로젝트는 사용자의 선택과 응답을 단계적으로 수집하여  
적합한 메뉴를 추천하고, 추천 결과에 대한 피드백(만족/불만족)에 따라  
**외부 API 및 LLM과 연계된 후속 흐름**을 제공하는 것을 목표로 합니다.

## 시스템 구조

### Client (Flutter)
- 사용자 입력(선택/질문/자연어)을 수집하는 UI
- 추천 진행 상태 및 결과 표시
- 만족/불만족에 따른 분기 UI 제공
- 위치 정보 제공(식당 검색 단계)

### Server (Node.js / Express)
- 추천 세션 생성 및 상태 관리
- 질문 선택 및 응답 처리
- 추천 결과 생성
- 외부 API 연동(Places, LLM)
- 모든 비즈니스 로직을 서버에서 처리

### Database
- 메뉴 데이터(태그 기반)
- 질문 데이터
- 추천 세션 및 진행 상태
- 사용자 피드백 이벤트 로그

---

## 시스템 설계

### 1. 질문 기반 추천 구조
- 메뉴는 국가/맛/형태 등의 **태그 조합**으로 표현
- 질문은 특정 태그를 판별하기 위한 단위로 정의
- 서버는 현재 후보군을 가장 잘 분리하는 질문을 선택하여 반환

### 2. 결과 분기 구조
- 추천 결과 제시 후 사용자 피드백을 기준으로 분기
  - 만족: 외부 Places API를 통한 식당 탐색
  - 불만족: 자연어 입력 기반 재추천 단계로 이동

### 3. LLM 연계 구조
- LLM 결과는 내부 메뉴 데이터와 매칭하여 검증 후 사용

---

## 기술 스택
- **Mobile:** Flutter (Dart)
- **Backend:** Node.js (Express)
- **Database:** MongoDB (문서 기반 데이터 모델)
- **LLM:** Gemini API (서버 연동)
- **Places:** 외부 장소 검색 API (카카오/네이버/구글 중 택)
